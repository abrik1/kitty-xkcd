# kitty-xkcd
view xkcd comics in kitty

# requirements:
- kitty
- curl (to download images)
- BSD/Linux (guh bindows) 

# features:
- [x] caches images
- [x] fast
- [x] uses kitty's icat to preview images
- [x] no 3rd party dependencies needed
